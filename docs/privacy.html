Privacy Policy

Last updated: June 2026

1. Information We Collect

Our applications are built with user security as our top priority. We do not collect, store, or transmit any personal identifying information or telemetry data to external third-party servers.

2. Data Encryption and On-Device Storage

Any application data configuration, security values, or personal preferences are kept isolated within the user sandbox environment. Secure data parameters utilize the native iOS hardware-level Keychain Services for system encryption storage, ensuring no tracking profile visibility.

3. Contact Details

If you have any operational queries regarding our ecosystem protocols, reach out via official campus channels at the Apple Developers Group, MIT Manipal.
